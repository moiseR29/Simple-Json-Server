# SERVER CHALLENGE

## how to use?

1. Install dependencies
2. execute 
  - ```bash
      npm start
    ```

## API
 
### all products

```bash
curl http://localhost:3000/data
```

### product by id

```bash
curl http://localhost:3000/data?_id=001
```

### product by xxxx

```bash
curl http://localhost:3000/data?key=value
```
